# Digital Menucard
This system stores drink names with their prices and allows the user to check if a drink is available.
It also demonstrates how to add new items and update existing prices inside a dictionary.

## Features
- Stores drinks and prices in a dictionary
- Prints the price of a specific drink (“water”)
- Adds a new drink to the menucard
- Updates the price of an existing drink
- Checks if a searched drink exists
- Shows the price if found
- Gives a clear message if the drink is not on the menu

# How to use
Run the system and type the name of any drink you want to check.
The program will display the price or tell you if the drink is unavailable.


## Learning Purpose
This project was created as part of backend engineering practice.
The goal was to improve skills in:
- dictionaries
- updating and adding data
- conditional logic
- input handling
